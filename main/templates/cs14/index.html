This is a site designed to test your programming knowledge for jobs related to Avaloq. This will be done via a variety of small programming problems of varying difficulty. Each problem will have test cases that can be run against it that will test how well your solution solves the problem. These can be run an infinite number of times. You may also create your own test cases to test against your solution. The problems that are assigned to you can be viewed on your profile page which is here

Tutorial on Using the Site

Profile page

This is the page where you will go to acsess the specific taks that are assigned to you and where you can view results from previous tasks that you have completed. An example of this page is shown below:

The page shows a list of the tasks that you need to complete and will tell you the amount of time you have been given for each task.
Clicking on start task will take you to the coding page for the task which looks like the example below.

Coding Page

The green timer at the top will show you how much time you have left to do this challenge before it is auto submitted.
The selector can be used to change the language the code is written in. Currently the site supports Java and Python.
The run button will run the code and give output in the "output" box.


Putting data into the custom input box and ticking the select box for custom input will allow you code to take in custom input and test with that with results again showing in the output box.
The submit button is used to submit the final version of your code.


Once the code is submitted you will be taken to the results page which will look similar to below.

Results Page

This will show all the different coding tests you have completed. In a list. Cicking the view results button will show that submissions results in a table as seen below.


This table shows details about the results for the test. Including the tests passed, tests failed, time taken to complete the tests. The complexity of the code and the description of the task.
There is also a button to view the code submitted which will take you to a page like this.

Review Page

This page works almost exactly the same as the coding page. You will be able to review your past code. The green bar at the top is a timeline.
Moving the bar to the right will show later versions of the code and moving it to the left will show earlier versions of the code. (A copy of the code is saved each time it is run)
This code can be modified for example to test it with syntax fixes. None of these changes will be saved though.
